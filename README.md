This repo was originally a Mathematics In Lean repo that I forked in order to
easily get all of the Mathlib Dependencies needed for my formulations. After
forking it I deleted all of the Mathematics In Lean things. All of my code for
the coding part of this project is in the FinalProject.lean file. Originally, the
scope of the project was to just formulate the first theorem you see on there but
it turned out much shorter than I expected so I ended up adding some more lemmas
that are either closely related or directly build off of the first one to make the
project a little bit more comprehensive.



--------------------------- AI USAGE AND OTHER CITATIONS ---------------------------
I mainly used AI in order to help me set up and navigate the codespace in the first
place since I was not used to using github codespaces. Other than that, I used it in
order to gain a surface level understanding on the strategies that I should use for
formalizing each of the lemmas and how some of them could rely on previous lemmas in
order to make the formalization process shorter.

I also used the following websites in order to learn more about specific tactics or
to check what is already out there in terms of formalizations:

https://leansearch.net/
https://leanprover-community.github.io/

I also used https://live.lean-lang.org/ for a while at the beginning of the formalization
process before I eventually ended up learning how to set up and use the codespaces.
